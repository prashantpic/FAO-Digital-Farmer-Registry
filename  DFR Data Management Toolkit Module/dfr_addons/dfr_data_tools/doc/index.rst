############################
DFR Data Management Toolkit
############################

Introduction
============

The DFR Data Management Toolkit (`dfr_data_tools`) is an Odoo module designed to provide administrators with comprehensive tools for managing data within the Digital Farmer Registry (DFR) system. Its primary features include bulk import capabilities for legacy farmer data from CSV and XLSX files, integrated data validation, detailed logging of import processes, and guidance on leveraging Odoo's standard export functionalities for DFR data. This toolkit is essential for data migration activities and ongoing data management tasks, ensuring data integrity and operational efficiency.
